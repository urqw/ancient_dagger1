# Ancient Dagger 1

Text-based game "Ancient Dagger 1" (Russian: "Древний кинжал 1") by Kaschey (Russian: Кащей) and RipOs.

The game is in Russian, developed for RipURQ, and adapted for [UrqW](https://github.com/urqw/UrqW) text-based game engine.

This repository contains the source code and other resources of the game. The game is available for launch in the [UrqW engine online catalog](https://urqw.github.io/UrqW/#ancient_dagger1).

The game was first published on December 9, 2000.

Interactive fiction identifier (IFID) of the game which is its MD5 hash: B532D767EC46DA0EE4DA2A17DFCB1A8B

## Description

Наивный, ещё ничего не подозревающий хоббит Фред (прямой потомок Фродо), сидел как-то на завалинке своего дома и лениво размышлял, чем бы заняться дальше. "Уж не отремонтировать ли поместье?" - взбрело ему в голову. У нашего героя слово не расходится с делом: не прошло и месяца как Фред потянулся к молотку. И как он мог быть так беспечен? Ведь все знают, как опасна может быть простая уборка, не говоря уже о ремонте. А уж к каким последствиям это может привести...
